## Springboard Data Science Assignments
This repository contains a curated collection of course assignments and applied exercises completed as part of the Springboard Data Science Career Track.
Each notebook focuses on practical problem-solving using real datasets and industry-standard tools.
The goal of this repository is to demonstrate:
- Structured analytical thinking
- Proficiency with Python, SQL, and Spark
- End-to-end machine learning workflows
- Clear documentation and reproducible analysis
